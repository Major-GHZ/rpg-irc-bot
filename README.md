# Lua RPG Bot Game

[Français](#en-français) | [English](#in-english)

---

## In English

### Description
This is a **team-based RPG game** played with a bot. The game is written in **Lua** and designed to be integrated with IRC or other chat platforms.

### Features
- Turn-based combat system
- Team play with multiple characters
- Lua-based, easy to modify and extend

### Installation
1. Clone this repository:
   ```bash
   git clone https://github.com/your-username/lua-bot.git
   ```
2. Ensure you have Lua installed (version 5.4+ recommended).
3. Run the game with:
   ```bash
   lua main.lua
   ```

### Usage
- Customize characters, skills, and game logic in the Lua files.
- Integrate with your preferred chat bot (e.g., IRC, Discord).

### Contributing
Pull requests are welcome! For major changes, please open an issue first.

## En Français

### Description
Ceci est un **jeu de RPG en équipe** qui se joue avec un bot. Le jeu est écrit en **Lua** et conçu pour être intégré à des plateformes de chat comme IRC.

### Fonctionnalités
- Système de combat au tour par tour
- Jeu en équipe avec plusieurs personnages
- Basé sur Lua, facile à modifier et à étendre

### Installation
1. Clone ce dépôt :
   ```bash
   git clone https://github.com/ton-utilisateur/lua-bot.git
   ```
2. Assure-toi d'avoir Lua installé (version 5.4+ recommandée).
3. Lance le jeu avec :
   ```bash
   lua main.lua
   ```

### Utilisation
- Personnalise les personnages, les compétences et la logique du jeu dans les fichiers Lua.
- Intègre-le avec le bot de chat de ton choix (ex: IRC, Discord).

### Contribution
Les pull requests sont les bienvenues ! Pour les changements majeurs, ouvre d'abord une issue.
